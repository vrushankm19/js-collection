speed: This parameter controls the speed of the expand and collapse animation in milliseconds. For example, speed: 500 sets the animation speed to 500 milliseconds (half a second).

collapsedHeight: This parameter determines the maximum height (in pixels) at which the content is displayed when it's collapsed. When the content exceeds this height, it gets truncated, and the "Read More" link appears. For example, collapsedHeight: 100 would truncate the content to a maximum height of 100 pixels.

moreLink: This parameter allows you to specify the HTML for the "Read More" link. You can customize the text and appearance of the link. For example, moreLink: '<a href="#">Read More</a>' sets the link text to "Read More" and creates an anchor element.

lessLink: Similarly, this parameter lets you specify the HTML for the "Read Less" link. You can customize the text and appearance of this link as well. For example, lessLink: '<a href="#">Read Less</a>' sets the link text to "Read Less" and creates an anchor element.

embedCSS: When set to true, this parameter includes the default CSS styles for "readmore.js." If set to false, it won't include the default styles, allowing you to define your own CSS styles for the content. By default, it's set to true.

blockCSS: This parameter allows you to specify additional CSS styles that will be applied to the content block when it's expanded. For example, you can use it to set a background color, font size, or padding for the expanded content.

startOpen: If set to false, the content will initially be collapsed, and users will need to click "Read More" to expand it. If set to true, the content will be expanded by default when the page loads.
